About Me

My name is Moises C. Dobarganes, most people call me “Moe” and I am a software developer. I graduated from Full Sail University in 2004 and am currently enrolled in Lambda School’s computer science course. I have worked as Graphic Designer, Computer Animator, Game Designer, IT Solutions Consultant, Operations Manager, and Software Developer. Over the years I have developed a passion for programming and solving problems through leveraging computing power. I enjoy doing data analysis and visualization, as well as coding user interfaces and predicting user patterns. My favorite part of developing software lies in pre-production and planning, I like whiteboarding an Idea, designing the different aspects of the program, and how those modules interact with each other.

Common JS Functions Broken Down (JS)

This project tackles breaking down commonly used JavaScript functions that are built in either within arrays as prototypes or any other data types. The project began as an attempt to better understand the innerworkings of common JavaScript functionality, and it keeps growing. This helps developers who are beginning to code get a better understanding of what is going under the hood on some of those prototype functions.
